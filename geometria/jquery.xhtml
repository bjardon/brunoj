Cálculo del área de un cuadrado

Se realiza mediante la fórmula l², siendo l el tamaño de los lados de un cuadrado.



Cálculo del área de un rectángulo

Se realiza mediante la fórmula b * h, donde b es el tamaño de la base y h es la altura del rectángulo.




Cálculo del área de un triángulo

Se realiza mediante la fórmula (b * h) / 2, donde b es el tamaño de la base y h es la altura del triángulo.




Cálculo del área de un círculo

Se realiza mediante la fórmula pi * r², donde r es el radio del círculo.



Cálculo del área de un trapecio

Se realiza mediante la fórmula ((b + B) / 2) * h, donde b es la base menor y B es la base mayor y h es la altura.





Cálculo del área de un rombo

Se realiza mediante la fórmula (d * D) / 2, donde d es la diagonal menor y D es la diagonal mayor.




Cálculo del área de un polígono regular

Se realiza mediante la fórmula (nl * a * l) / 2, donde nl es el número de lados, a es el apotema y l es el tamaño del lado.
