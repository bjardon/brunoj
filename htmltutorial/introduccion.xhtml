Introducción a HTML

HTML es un lenguaje de marcado de hipertexto. Esto significa que es un lenguaje en el que se utilizan etiquetas para darle un significado a las palabras denntro de un contexto semántico determinado.

Desde sus inicios, el internet siempre ha sido eso: texto con links, que llevan a más texto con links. Antes de poder adentrarnos en el desarrollo de aplicaciones y construir la nueva red social que desbancará a Facebook, tenemos que dominar (o por lo menos conocer) el funcionamiento baśico de las páginas web tradicionales, aquellas que solamente tenían un montón de texto y algunos enlaces.

Documentos HTML

En el desarrollo web, a las páginas se les conoce como "documentos"; esto porque cada página tiene un propósito y un significado propio. Los documentos HTML están compuestos por texto plano que contiene etiquetas para definir la estrucutura de la página, así como la información que se presentará en esta.

La estructura básica de un documento HTML5 es la siguiente:

<!DOCTYPE html>
<html>
<body>
 <h1>My First Heading</h1>
 <p>My first paragraph.</p>
</body>
</html>

Semántica HTML

Como ya dijimos antes, una página es un documento; por lo tanto tiene una semántica. Y como tiene una semańtica, se necesita de una estructura que defina esta semántica.

Dentro de HTML contamos con cuatro elementos básicos:

  • Encabezados
  • Párrafos
  • Enlaces (links)
  • Imágenes

Estos elementos nos sirven para describir una página web extremadamente sencilla, donde se cubra el objetivo principal de hacer una página web: comunicar información y enlazar con más información.

Aunque en este artículo no se pretende explicar detalladamente cada uno de estos elementos, a continuación están explicados brevemente.

Encabezados

Los encabezados nos sirven para definir títulos en una estructura jerárquica. Contamos con seis niveles: desde <h1> hasta <h6>.

Párrafos

Como cualquier otro documento de texto, llámese PDF o de MS Word, el texto se agrupa en párrafos

Enlaces o links

Son vínculos que se establecen para referenciar a otros recursos; ya sea locales (en el mismo servidor), remotos (en otros servidores) o dentro del mismo documento.

Imágenes

Por otro lado, podemos incluir imágenes para acompañar nuestro texto simplemente referenciando el archivo de imagen que se desea mostrar.
